3D reconstruction by NeRF and SfM from 360Â° camera images [Paper]

Capturing a scene with Lidar inherently needs a large amount of data storage. This study investigates alternative approaches to capture 3D information via 360 videos using Structure-from-Motion (SfM) and Neural Radiance Fields (NeRF).

Semi-supervised symbols detection in Piping and Instrumentation Drawings (P&IDs) [Paper under review]

Typically, P&IDs have many symbols depicting various equipment. Labeling symbols is challenging. A semi-supervised pipeline was built for symbol detection achieving a Top-1 accuracy of 85.49% over 102 test classes.

Automated 2D building plans to 3D digital model Journal paper; Conf paper

This project aims to speed up the conversion of 2D CAD into 3D BIM models. A semantic segmentation-based approach was found to be more performant than bounding-based methods, with linkage to 3D drafting software via Dynamo API.

Valve detection in P&IDs without human annotations: A synthetic data approach [Conf paper]

A synthetic data generation pipeline was devised to create a labeled dataset without human annotations. The presence of look-alike symbols (hard negatives) helped reduce false positives and increased performance.
