Semi-supervised symbols detection in Piping and Instrumentation Drawings


Developed a Siamese Network for symbol retrieval and applied K-means coreset sampling to achieve nearly the same performance as using the full dataset, but with only 30% of the data.
Improved the Top-1 accuracy by 4.5% with Ensemble learning.

Kaggle DFL Bundesliga Data Shootout


Implemented an LSTM network achieving 76% accuracy to recognize different types of soccer passes in Football tournament videos.

Automated 2D building plans to 3D digital model


Trained a Yolov3 model to recognize building components in 2D CAD. Then, linked the localised coordinates with Dynamo to create 3D model in Revit.

Valve detection in P&IDs without human annotations: A synthetic data approach


Devised a synthetic-data generation pipeline to create a labelled dataset without any human annotations. Presence of a look-alike symbols (i.e. hard negatives) reduced false positives & increased the performance.
